# Documentação de Regras - AuditPlus

Esta pasta contém a documentação detalhada de todas as regras implementadas no sistema AuditPlus, incluindo:

- **Fundamentação normativa** (boletins, manuais PTU)
- **Lógica de aplicação**
- **Códigos afetados**
- **Exemplos de casos corrigidos**

## Estrutura dos Documentos

Cada regra possui um arquivo markdown com:
1. Identificador único da regra
2. Data de criação e última atualização
3. Fundamentação legal/normativa
4. Descrição da lógica
5. Condições de aplicação
6. Ação corretiva
7. Exemplos reais (quando aplicável)
